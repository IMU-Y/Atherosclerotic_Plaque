说明：
1. dataset_split.py 为分割数据集代码，将数据集分为训练集验证集和测试集
2. augmentation.py 为数据集增强代码，选择4个序列共1500张图像作为测试集，剩余的11个序列共4124张图像，其中450张图像增强为450*8=3600张图像，
剩余的3674张图像增强为3674*8=23932张图像作为训练集
3. train_plaque_pause.py 为训练和验证代码
4. test_plaque.py 为测试代码
5. model 为网络模型文件夹
6. utils 为工具包 如加载数据集、损失函数
7. dataset.zip 为数据集，包含15个病人的oct序列，共5624张图像，解压后将数据放在一个新建的data文件夹钟


说明：可以实验一下编码器解码器的分割网络等，如FCN等
